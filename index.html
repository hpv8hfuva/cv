  • Movie Fight Night

    • Using HTML, CSS, and Javascript, Movie Fight Night is a movie comparison website. The user is able to search for specific movies, and this search will grab information found from the Open Movie Database (OMDb API). The website allows the user to compare two movies to determine which one is better based on statistics found within the API. If a user's search is too broad, the website displays an array of movie selections that closely relates to the searched title. The winning statistics on a movie is highlighted in aqua while the losing statistics are highlighted in yellow to help the user distinguish the better movie.
